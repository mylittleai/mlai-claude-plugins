# Contributing

Thanks for your interest. Please read this before contributing.

## How to contribute

1. **Open an issue first.** Describe what you want to change and why. This is the place to discuss ideas, report bugs, and ask questions.
2. **Wait for a response.** We'll discuss the approach before any code is written.
3. **Then submit a PR** if we agree on the direction.

Do not fork and open a pull request without discussing it in an issue first. Unsolicited PRs will be closed.

## Bug reports

Open an issue in this repo. Each plugin also has a `/feedback` command that files an issue here automatically:

- `mtplan` — run `/mtplan:feedback`
- `glass` — run `/glass:feedback`
